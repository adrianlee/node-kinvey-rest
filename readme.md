# node-kinvey-rest
A Kinvey REST API Client for node.js

## API Coverage
### Implemented
**Appdata**

+ GET /appdata/:appKey

**Users**

+ POST /user/:appKey/
+ GET /user/:appKey/:id
+ PUT /user/:appKey/:id
+ POST /user/:appKey/login
+ POST /user/:appKey/_logout
+ DELETE /user/:appKey/:id
